# Supported trigger patterns
## Ford ST170 Pattern
### Overview

Ford ST170 trigger pattern consists of 36-1 [missing_tooth](/en/decoders/Missing_Tooth) pattern on crank and 5-tooth cam trigger wheel for sequential operation and VVT.
